Building an agentic research tool for internships in India and remote roles is a brilliant project. The biggest hurdle with job/internship data is that the major platforms (like LinkedIn, Indeed, and Naukri) have completely locked down their public, free APIs.

To make your agent work without hitting a paywall, you have to target **global remote aggregators**, **freemium developer tools**, or **structured open feeds**.

The best free public APIs you can leverage for your agent include:

---

## 1. Remote-First Public APIs (Great for Remote Options)

Because these platforms focus entirely on remote work, their APIs are completely public, require no authentication, and will give your agent clean JSON data to parse.

### Adzuna API

* **What it is:** A massive job search aggregator that has excellent coverage for India.
* **The Free Tier:** They offer a generous free public tier (requires a free developer App ID and Key).
* **Why it works for your agent:** You can explicitly pass parameters like `country=in` (India) and text queries like `"internship"` or `"remote"`.

### We Work Remotely (WWR) API

* **What it is:** The largest remote work community globally.
* **The Free Tier:** 100% Free, no API key required.
* **Endpoint:** `[https://weworkremotely.com/api/v1/posts](https://weworkremotely.com/api/v1/posts)`
* **Why it works for your agent:** It returns a massive JSON/RSS feed of the latest remote jobs. Your agent will need to write a simple keyword filter over the payload to pull out titles containing `"Intern"`, `"Junior"`, or `"Graduate"`.

### Remotive API

* **What it is:** Another premier remote job board focused heavily on Tech, Marketing, and Design.
* **The Free Tier:** 100% Free, no auth required.
* **Endpoint:** `[https://remotive.com/api/remote-jobs](https://remotive.com/api/remote-jobs)`
* **Why it works for your agent:** It allows category filtering. Your agent can query it and parse the `location` field for "Worldwide" or "India".

---

## 2. Aggregators & Multi-Source Tool APIs (Best for India Coverage)

To get specific, localized Indian internships (like those in Bangalore, Mumbai, or Delhi), you need aggregators that scrape local ATS (Applicant Tracking Systems) data.

### TheirStack API

* **What it is:** A tool that tracks job postings across 300k+ sources and deduplicates them.
* **The Free Tier:** Offers a free tier of **200 credits per month**.
* **Why it works for your agent:** Unlike generic search engines, TheirStack lets your agent filter heavily by specific criteria—meaning you can pinpoint roles explicitly tagged as `country: "India"` and `employment_type: "internship"` without wasting API calls.

### SerpApi (Google Jobs API)

* **What it is:** Google for Jobs is arguably the best aggregator of Indian internships because it indexes platforms like Internshala, LinkedIn India, and corporate career pages automatically. SerpApi scrapes this structured data for you.
* **The Free Tier:** **100 free searches per month**.
* **Why it works for your agent:** Because your agent is doing targeted research, 100 highly optimized queries (e.g., searching specifically for *"Software Engineer Intern India"*) can pull back hundreds of live results via structured JSON.

### Arbeitnow API

* **What it is:** A tech job board aggregator pulling direct data from major company ATS pipelines (Greenhouse, Lever, etc.).
* **The Free Tier:** 100% Free, no API key required.
* **Endpoint:** `[https://www.arbeitnow.com/api/job-board-api](https://www.arbeitnow.com/api/job-board-api)`
* **Why it works for your agent:** It includes dedicated boolean flags for `remote` and `visa_sponsorship` which your agent can use to instantly classify global remote options that accept international talent.

---

## 💡 Pro-Tips for Your Agent's Architecture

1. **The Keyword Fallback Strategy:** Because many of these free APIs bundle internships into standard "Junior" or "Entry-level" data streams, instruct your agent's LLM core to act as a **post-query filter**. Have it fetch the payload from WWR or Remotive, and use regex or a light token check to map titles containing `["intern", "apprentice", "stipend", "co-op"]`.
2. **Handle Rate Limits Elegantly:** Since you're utilizing free tiers, implement a caching mechanism (like a lightweight SQLite database or Redis) so your agent doesn't re-query the same API endpoint multiple times during a single user research session.

import requests
import time
from config.settings import Settings

class HimalayasAgent:
    def __init__(self):
        self.api_url = Settings.HIMALAYAS_API_URL
        self.rate_limit = Settings.HIMALAYAS_RATE_LIMIT
        self.request_count = 0
        self.last_request_time = 0
    
    def _check_rate_limit(self):
        """Check if we've hit rate limit"""
        if self.request_count >= self.rate_limit:
            wait_time = 3600  # Wait 1 hour
            time.sleep(wait_time)
            self.request_count = 0
    
    def fetch_remote_internships(self, limit=20, offset=0):
        """
        Fetch remote internships from Himalayas
        Verified endpoint: https://himalayas.app/jobs/api
        """
        self._check_rate_limit()
        
        params = {
            "limit": limit,
            "offset": offset
        }
        
        try:
            response = requests.get(self.api_url, params=params, timeout=10)
            self.request_count += 1
            self.last_request_time = time.time()
            
            if response.status_code == 200:
                data = response.json()
                return self._parse_internships(data)
            else:
                print(f"Himalayas API error: {response.status_code}")
                return []
        
        except requests.exceptions.RequestException as e:
            print(f"Himalayas request failed: {e}")
            return []
    
    def _parse_internships(self, data):
        """Parse Himalayas API response"""
        internships = []
        
        for job in data.get("jobs", []):
            internship = {
                "title": job.get("title"),
                "company": job.get("company", {}).get("name"),
                "location": job.get("location", "Remote"),
                "type": job.get("type"),
                "url": job.get("url"),
                "posted_date": job.get("posted_at"),
                "description": job.get("description"),
                "source": "Himalayas"
            }
            internships.append(internship)
        
        return internships
```

            "app_key": self.app_key,

            "location": location,

import requests
from bs4 import BeautifulSoup
import json

class InternshalaScraper:
    def __init__(self):
        self.base_url = "https://internshala.com"
        self.headers = {
            "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
        }
    
    def fetch_internships(self, page=1, limit=20):
        """
        Scrape Internshala directly (Unofficial - may break)
        Use Apify version if possible
        """
        url = f"{self.base_url}/internships/?page={page}"
        
        try:
            response = requests.get(url, headers=self.headers, timeout=10)
            
            if response.status_code == 200:
                soup = BeautifulSoup(response.text, 'html.parser')
                return self._parse_internships(soup)
            else:
                print(f"Internshala scrape failed: {response.status_code}")
                return []
        
        except Exception as e:
            print(f"Scraper error: {e}")
            return []
    
    def _parse_internships(self, soup):
        """Parse Internshala HTML"""
        internships = []
        
        job_cards = soup.find_all('div', class_='internship_listing')
        
        for card in job_cards:
            title_elem = card.find('a', class_='listing-parent')
            company_elem = card.find('span', class_='company-name')
            location_elem = card.find('span', class_='location')
            stipend_elem = card.find('span', class_='stipend')
            
            if title_elem:
                internship = {
                    "title": title_elem.get_text(strip=True),
                    "company": company_elem.get_text(strip=True) if company_elem else None,
                    "location": location_elem.get_text(strip=True) if location_elem else None,
                    "stipend": stipend_elem.get_text(strip=True) if stipend_elem else None,
                    "url": title_elem.get('href'),
                    "source": "Internshala (Scraper)"
                }
                internships.append(internship)
        
        return internships

        """
        Merge multiple internship lists and clean data
        all_internships: List of lists from different agents
        """
        # Flatten all lists
        merged = []
        for internship_list in all_internships:
            merged.extend(internship_list)
        
        if not merged:
            return pd.DataFrame()
        
        df = pd.DataFrame(merged)
        
        # Standardize columns
        df = self._standardize_columns(df)
        
        # Remove duplicates (same title + company + url)
        df = df.drop_duplicates(
            subset=['title', 'company', 'url'],
            keep='first'
        )
        
        # Remove empty entries
        df = df[df['title'].notna() & df['url'].notna()]
        
        # Sort by posted date (newest first)
        if 'posted_date' in df.columns:
            df['posted_date'] = pd.to_datetime(df['posted_date'], errors='coerce')
            df = df.sort_values('posted_date', ascending=False)
        
        return df
    
    def _standardize_columns(self, df: pd.DataFrame) -> pd.DataFrame:
        """Standardize column names across sources"""
        column_mapping = {
            'title': 'title',
            'company': 'company',
            'location': 'location',
            'type': 'job_type',
            'url': 'url',
            'posted_date': 'posted_date',
            'description': 'description',
            'source': 'source'
        }
        
        # Rename to standard names
        df = df.rename(columns={k: v for k, v in column_mapping.items() if k in df.columns})
        
        # Fill missing columns with None
        for col in column_mapping.values():
            if col not in df.columns:
                df[col] = None
        
        return df
    
    def filter_by_location(self, df: pd.DataFrame, location: str) -> pd.DataFrame:
        """Filter internships by location (e.g., 'Remote', 'India', 'Bhopal')"""
        if location.lower() == 'remote':
            return df[df['location'].str.lower().contains('remote', na=False)]
        else:
            return df[df['location'].str.contains(location, na=False, case=False)]
```

import pandas as pd
import json
from pathlib import Path
from config.settings import Settings

class FileHandler:
    def __init__(self):
        self.output_dir = Path(Settings.OUTPUT_DIR)
        self.output_dir.mkdir(exist_ok=True)
    
    def save_to_csv(self, df: pd.DataFrame, filename: str = None):
        """Save DataFrame to CSV"""
        if filename is None:
            filename = Settings.CSV_FILE
        
        filepath = self.output_dir / filename
        df.to_csv(filepath, index=False, encoding='utf-8')
        print(f"✓ Saved to: {filepath}")
        return filepath
    
    def save_to_json(self, df: pd.DataFrame, filename: str = None):
        """Save DataFrame to JSON"""
        if filename is None:
            filename = Settings.JSON_FILE
        
        filepath = self.output_dir / filename
        df.to_json(filepath, orient='records', forcebuf=False, ensure_ascii=False)
        print(f"✓ Saved to: {filepath}")
        return filepath
    
    def save_to_excel(self, df: pd.DataFrame, filename: str = "internships.xlsx"):
        """Save DataFrame to Excel with formatting"""
        filepath = self.output_dir / filename
        df.to_excel(filepath, index=False)
        print(f"✓ Saved to: {filepath}")
        return filepath

## 🎯 Step 8: Main Entry Point

### `main.py`

```python
import argparse
from agents.himalayas_agent import HimalayasAgent
from agents.adzuna_agent import AdzunaAgent
from agents.internshala_agent import InternshalaAgent
from utils.data_cleaner import DataCleaner
from utils.file_handler import FileHandler

class InternshipResearchAgent:
    def __init__(self):
        self.himalayas = HimalayasAgent()
        self.adzuna = AdzunaAgent()
        self.internshala = InternshalaAgent()
        self.cleaner = DataCleaner()
        self.file_handler = FileHandler()
    
    def research_all_sources(self, location="india", remote_only=False):
        """
        Research internships from all sources
        """
        print("🚀 Starting internship research...\n")
        
        all_internships = []
        
        # 1. Fetch from Himalayas (Remote)
        if remote_only:
            print("📍 Fetching remote internships from Himalayas...")
            remote_jobs = self.himalayas.fetch_remote_internships(limit=50)
            all_internships.append(remote_jobs)
            print(f"✓ Found {len(remote_jobs)} remote internships\n")
        
        # 2. Fetch from Adzuna (India)
        print("📍 Fetching India internships from Adzuna...")
        if location.lower() == "india":
            india_jobs = self.adzuna.fetch_india_internships(keyword="internship", results=100)
        else:
            india_jobs = self.adzuna.search_by_city(city=location)
        all_internships.append(india_jobs)
        print(f"✓ Found {len(india_jobs)} India internships\n")
        
        # 3. Fetch from Internshala (India-specific)
        print("📍 Fetching Internshala internships...")
        internshala_jobs = self.internshala.fetch_internshala_internships(limit=50)
        all_internships.append(internshala_jobs)
        print(f"✓ Found {len(internshala_jobs)} Internshala internships\n")
        
        # Clean and merge
        print("🧹 Cleaning and merging data...\n")
        df = self.cleaner.clean_and_merge(all_internships)
        
        if df.empty:
            print("❌ No internships found.")
            return
        
        print(f"🎯 Total unique internships: {len(df)}\n")
        
        return df
    
    def save_results(self, df: pd.DataFrame, format="csv"):
        """Save results to file"""
        if format == "csv":
            self.file_handler.save_to_csv(df)
        elif format == "json":
            self.file_handler.save_to_json(df)
        elif format == "excel":
            self.file_handler.save_to_excel(df)
    
    def display_summary(self, df: pd.DataFrame):
        """Display summary of results"""
        print("\n" + "="*60)
        print("📊 INTERNSHIP SUMMARY")
        print("="*60)
        
        print(f"\nTotal internships found: {len(df)}")
        
        # By source
        if 'source' in df.columns:
            print("\nBy Source:")
            source_counts = df['source'].value_counts()
            for source, count in source_counts.items():
                print(f"  - {source}: {count}")
        
        # By location type
        if 'location' in df.columns:
            remote_count = len(df[df['location'].str.lower().contains('remote', na=False)])
            print(f"\nRemote internships: {remote_count}")
            print(f"On-site internships: {len(df) - remote_count}")
        
        print("\n" + "="*60)
        
        # Show top 10
        print("\n🔥 TOP 10 INTERNSHIPS:")
        print(df.head(10)[['title', 'company', 'location', 'source']].to_string())

def main():
    parser = argparse.ArgumentParser(description="Internship Research Agent")
    parser.add_argument(
        "--location", 
        type=str, 
        default="india",
        help="City or 'india' for nationwide search"
    )
    parser.add_argument(
        "--remote-only",
        action="store_true",
        help="Search only remote internships"
    )
    parser.add_argument(
        "--format",
        type=str,
        default="csv",
        choices=["csv", "json", "excel"],
        help="Output file format"
    )
    
    args = parser.parse_args()
    
    # Run research
    agent = InternshipResearchAgent()
    df = agent.research_all_sources(
        location=args.location,
        remote_only=args.remote_only
    )
    
    if df is not None and not df.empty:
        # Display summary
        agent.display_summary(df)
        
        # Save results
        agent.save_results(df, format=args.format)

if __name__ == "__main__":
    main()
```
