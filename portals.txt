For a **unified internship aggregation platform**, don't think in terms of "top portals". Think in terms of **data source categories**.

A practical 2026 architecture would have ~25–40 sources.

# Tier A — Core Sources (80% of internship volume)

These should be your first integrations.

| Source         | Type                      | Priority |
| -------------- | ------------------------- | -------- |
| Internshala    | Internship-only           | P0       |
| Unstop         | Internship + competitions | P0       |
| LinkedIn Jobs  | Jobs + internships        | P0       |
| Naukri         | Jobs + internships        | P0       |
| Foundit        | Jobs + internships        | P0       |
| Wellfound      | Startup internships       | P0       |
| Apna           | Jobs + internships        | P1       |
| Indeed India   | Jobs + internships        | P1       |
| Glassdoor Jobs | Jobs + internships        | P1       |

Internshala, Unstop, AICTE, LinkedIn, Naukri, Foundit, Wellfound collectively cover most mainstream internship postings in India. ([Unstop][1])

# Recommended MVP (First 10 Sources)

If you're building this project yourself:

```text
1. Internshala
2. Unstop
3. AICTE
4. Skill India
5. LinkedIn
6. Wellfound
7. Naukri
8. Foundit
9. Cutshort
10. Startup India
```

This alone can realistically give **50k–150k active internship listings** after deduplication. ([Unstop][1])

# Ideal Architecture

```text
Collectors
│
├── Portal Scrapers
├── Government APIs
├── Career Page Crawlers
├── RSS Feed Collectors
├── Sitemap Crawlers
└── Community Feed Parsers

        ↓

Raw Queue (Kafka)

        ↓

Normalizer

        ↓

Deduplication
(company + title + embedding)

        ↓

PostgreSQL

        ↓

Search Index
(Elasticsearch/OpenSearch)

        ↓

FastAPI

        ↓

Next.js Frontend
```

For long-term maintenance, I would prioritize **career-page crawling + AICTE + Unstop + Internshala** over trying to scrape everything. Those four sources provide the highest signal-to-maintenance ratio in India as of 2026. ([AICTE Internship Portal][2])
